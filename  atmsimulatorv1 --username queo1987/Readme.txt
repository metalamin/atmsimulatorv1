Demo user guide
---------------

A - ATM SIMULATOR
-----------------
To run ATM Simulator Demo follow this step:

1. execute: "runDemo.bat"

or if you have problems running it, please execute this three steps separately:

1. execute: "runJxfsServer.bat"
2. execute: "runServer.bat"
3. execute: "runAtmSimulator.bat"

B - SCREEN EDITOR
-----------------
To run Screen Editor App follow this step:

1. execute: "runScreenEditor.bat"

C - STATE EDITOR
----------------
To run State Machine Editor App follow this step:

1. execute: "runStateEditor.bat"
